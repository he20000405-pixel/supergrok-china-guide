# 图片素材说明

- 只使用真实页面截图或明确标注的虚构数据示意图。
- 不上传真实卡密、订单号、邮箱、User ID 或其他用户隐私。
- `grok-user-id-json-example.webp` 使用虚构 UUID 与隐藏邮箱，仅用于说明字段位置。
- 不保留固定价格截图；套餐与价格统一链接到 `https://chonggrok.com/supergrok` 实时页面。
- 新截图优化完成前，可在 Markdown 源码中用 HTML 注释记录占位，不在公开页面显示空白框。
